How it works in the actual code: One HTML file, one CSS block with :root for dark (default) and @media (prefers-color-scheme: light) { :root { ... } } to override. Zero JavaScript. The terminal snippet always stays dark regardless of mode — that's intentional and standard practice (GitHub does this too).
